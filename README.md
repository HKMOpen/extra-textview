## feature
- icon drawable support(`auto resize`, `specific size`)

## Download
### Gradle
1. add JitPack repository to your project root `build.gradle`.
```
repositories {
    maven { url "https://jitpack.io" }
}
```

2. add the dependency
```
dependencies {
    compile 'com.github.chuross.extra-textview:x.x.x'
}
```
